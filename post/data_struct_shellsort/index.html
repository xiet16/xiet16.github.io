特点:

本质上是一种分组
时间复杂度为O(n3/2)
可以用多线程

length:=len(arr)
for i:=start+gap;i<length;i+=gap {
	//间隔为gap 的插入排序
	backup :=arr[i]
	j:=i-gap
	//数据循环往后移动
	for j>=0 && backup>arr[j]{
		arr[j+gap] = arr[j]
		j-=gap
	}
	arr[j+gap]=backup //插入

}
